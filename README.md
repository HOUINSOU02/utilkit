# ⬡ UtilKit

Une suite d'outils pour développeurs **100% Client-Side** (Zero Server).
Rapide, léger, respectueux de la vie privée et doté d'une interface soignée (Mode Sombre / Clair).

## 🚀 Fonctionnalités

### 1. Générateur de QR Code
- Génération en temps réel.
- Personnalisation de la taille et des couleurs.
- Téléchargement direct en PNG.
- Support des thèmes (QR Code adapté au mode sombre/clair).

### 2. Outil Base64
- Encodage et décodage instantané.
- Support complet de l'UTF-8.
- Bouton "Swap" et copie rapide.

### 3. Générateur de Mots de Passe
- Cryptographiquement sécurisé.
- Calcul de l'entropie et de la force en temps réel.
- Options configurables (Majuscules, Chiffres, Symboles).
- Copie en un clic.

## 🛠 Installation & Démarrage

1. **Installer les dépendances**
   ```bash
   npm install
   ```

2. **Lancer le serveur de développement**
   ```bash
   npm run dev
   ```

3. Ouvrir `http://localhost:5173` dans votre navigateur.
